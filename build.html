สร้างด้วย

Chat Line Bot ที่ใช้ AI ในส่วนของ Deep learning และ Algorithm ประเภท Convolution Neural Network ซึ่งเหมาะสำหรับการประมวลผลข้อมูลชนิดภาพและ Line bot ที่สามารถเข้าถึงการใช้งานได้ง่ายและสามารถใช้ประโยชน์จากการให้ข้อมูลทางการรักษาโรคของมะเขือเทศที่อ้างอิงหลักการทางชีววิทยา และสามารถช่วยเกษตรกรในการป้องกัน การเกิดผลเสียของผลผลิตทางการเกษตรได้

คือ algorithm สำหรับการทำ Computer Vision ที่สำหรับ real-time ที่ถูกพัฒนามากจาก YOLO (You Only Look Once) ที่เป็นพื้นฐานเดิมอยู่แล้วโดยพัฒนาครั้งแรกด้วย YOLOv4 ต่อมาได้พัฒนามาเป็น YOLOv7

คือ Framework ที่เปิดสำหรับนักพัฒนาในด้าน Computer Vision เพื่อการรวบรวมข้อมูลที่สะดวก และสามารถเพิ่มข้อมูลได้รวดเร็ว

คือ แพลตฟอร์มปัญญาประดิษฐ์ (AI) ที่เกิดจากการร่วมมือระหว่างสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบังร่วมกับมหาวิทยาลัยขอนแก่น และมหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ
